Utilizando id

É utilizado para podemos selecionar apenas um seletor para modificar no CSS assim nos dando mais organização para o nosso codígo e mais modificação inedividual para o nosso website.

Utilizando classes

Segue o mesmo principior de id, ou seja, ter seletores para podemos modificar seletores de maneira individual, mas com as classes são um pouco diferente pois esta relacionada a um conjuntos, em multi-seletores.
